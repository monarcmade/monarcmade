## Proposed Monarc University Cohort Spine

### 12-Week Fast Track

| Weeks | Phase | Outcome |
|---|---|---|
| 1-2 | AI foundations and prompt systems | Understand LLMs, prompts, model behavior, and structured outputs |
| 3-4 | Workflow design and business use cases | Turn repetitive work into repeatable AI workflows |
| 5-6 | APIs and AI app foundations | Build simple AI-powered tools with real APIs |
| 7-8 | RAG and knowledge systems | Build assistants that use documents, SOPs, sites, and business data |
| 9-10 | Agents and automations | Build assistants that use tools and trigger workflows |
| 11-12 | Production and capstone | Ship a working AI tool, demo it, and package it as a portfolio/client asset |

### 24-Week Standard Track

Same phases as the fast track, but each phase gets four weeks instead of two:

- Week 1: concept
- Week 2: demo
- Week 3: lab
- Week 4: project review

## Offer Packaging Notes

The lighter intro offer can stay accessible, but the full cohort should be priced like a serious guided program.

Suggested ladder:

- Self-paced starter: $297-$497
- 12-week cohort: $1,497-$2,500
- 24-week cohort: $2,500-$4,000
- VIP / build-with-me: $5,000+

The more Monarc University includes live review, accountability, capstone feedback, implementation support, and business outcomes, the more confidently it can be priced.
